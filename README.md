News Tracer 🔍

新闻溯源和真实性验证工具

专为短线交易者和信息验证者设计，快速追踪新闻来源，验证真实性，避免被假新闻或夸大报道影响交易决策。

## ✨ 主要功能

- 📊 **新闻溯源**：追踪新闻的最早来源和完整传播链
- ⏱️ **时间线生成**：按发布时间排序，建立完整时间线
- 🔍 **真实性评估**：检查权威媒体报道，评估可信度
- ⚠️ **风险提示**：识别夸大报道、标题党，给出风险提示
- ✅ **交叉验证**：多家媒体报道对比，找出矛盾点
